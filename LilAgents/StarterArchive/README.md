# Pathpoint Starter Knowledge Base

A bundled starter pack of Pathpoint FAQs and appetite guides for use by Tiny Ellie, Pathpoint's AI assistant for retail insurance agents.

## What Is Included

- **10 FAQ articles** covering quoting, binding, PathPay, subjectivities, renewals, and billing
- **11 appetite guides** covering all of Pathpoint's primary product lines and verticals

## Folder Layout

```
StarterArchive/
├── index.json          ← searchable catalog of all content
├── faqs/               ← help center articles and how-to guides
└── appetite/           ← product appetite and coverage details
```

## Content Sources

- FAQs: help.pathpoint.com
- Appetite guides: pathpoint.com/state-guide and pathpoint.com product pages

## For the Full Knowledge Base

Connect to the Pathpoint MCP server in Tiny Ellie's Settings for access to the complete Pathpoint knowledge base.
